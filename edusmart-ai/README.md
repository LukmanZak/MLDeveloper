# AI Administrasi Guru

Solusi berbasis Artificial Intelligence untuk membantu tenaga pendidik mengelola administrasi sekolah secara otomatis, efisien, dan terstruktur.

jangan lupa konfigurasi gemini.ts nya ya..

## Fitur Utama
- Penyusunan Modul Ajar: Membuat RPP dan Modul Ajar Kurikulum Merdeka dalam hitungan detik.
- Generator Asesmen: Membuat bank soal dan rubrik penilaian secara otomatis.
- Otomasi Laporan: Membantu penyusunan narasi rapor dan evaluasi pembelajaran siswa.

## Jalankan Secara Lokal

Prasyarat: Node.js

1. Instal dependencies:
   ```bash
   npm install

2. Konfigurasi API Key:
Set GEMINI_API_KEY pada file .env.local dengan API key Gemini Anda:

Code snippet
GEMINI_API_KEY=your_api_key_here
Jalankan aplikasi:

3. Bash
npm run dev
Teknologi yang Digunakan
Framework: Next.js / React

AI Engine: Google Gemini SDK

Styling: Tailwind CSS
